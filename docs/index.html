About

The Library Management System is a modern, web-based application designed to streamline library operations. Built with Laravel and Blade templates, it provides a user-friendly interface for managing books, members, borrowing records, and more.

Whether you're running a small community library or a large academic institution, this system offers the tools you need to efficiently manage your library's resources.

Key Features

📖 Book Management

Add, edit, and manage your library's book collection with detailed information including ISBN, author, publication date, and availability status.

👥 Member Management

Maintain member records, track borrowing history, and manage membership status with ease.

🔄 Borrowing System

Track book borrowing and returns, manage due dates, and handle overdue book notifications.

📊 Reports & Analytics

Generate comprehensive reports on library usage, popular books, and member activity.

🔐 User Authentication

Secure login system with role-based access control for different types of users.

⚡ Responsive Design

Works seamlessly on desktop, tablet, and mobile devices for on-the-go access.

Getting Started

Prerequisites

  • PHP 8.0 or higher
  • Composer
  • MySQL 5.7 or higher
  • Git

Installation Steps

1. Clone the repository:

git clone https://github.com/Nafis878/Library-Management-System.git
cd Library-Management-System

2. Install dependencies:

composer install

3. Create environment file:

cp .env.example .env

4. Generate application key:

php artisan key:generate

5. Configure database in .env file:

DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=library_db
DB_USERNAME=root
DB_PASSWORD=

6. Run migrations:

php artisan migrate

7. Start the development server:

php artisan serve

The application will be available at http://localhost:8000
